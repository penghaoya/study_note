# 异地恋怎么谈才香？

## 如何判断异地恋能不能谈？

> 一看两个人能否具备共同的目标，二看对方在行为上和这个目标是否一致

::: warning

- 一看两个人能否达成共同的目标，
- 二看对方在行动上和这个目标是否一致。

也就是说但凡是这两个条件有一个不具备的话，而你又看重恋爱的效率，不想浪费精力。

那么我觉得大家是可以对这段关系去三思的，就这么两条规则，这可以去帮大家审视几乎所有的异地恋情况，

不管这段异地恋的困难具体是什么，其实最后最终都会反映在这两条规则上。正因为他们极其的高效并且简单，所以我就放在第一点讲了

严谨的说所有的恋爱都需要这两点,异地恋当中尤为重要

:::

### 目标一致

共同目标像是一种思想和纲领。一个清晰的思想和纲领确保两个人在同一方向努力。
::: tip 目标一致

一个清晰的思想和纲领会保证两个人在同一方向上努力，这会提高我们的效率，减少很多恋爱当中不必要的麻烦啊。
:::

::: warning 例子
两个人谈恋爱就像这么一个团队，而且还是远程办公。唉，中间的很多事情沟通不一定及时或者清晰，那么有了共同目标，两个人就相当于有了一个共同的行动指南。就说在很多细节的事情上，即便互相不说，各自也知道自己应该做什么啊，怎么做上市？两个人说好了三年后去领证结婚，这是共同目标，那么两个人的心里面都会自动的出现一种规划，为了实现这个目标，我们就会想应该去存钱，反过来也不能凭着喜好乱花钱了。我们会去和父母沟通这件事，会去预习婚礼应该怎么办？有哪些流程等等。
这像是一种无形的力量，把两个人，嗯在一个轨道上，让两个人都朝着那个终点的方向渐行渐进。反过来，如果没有共同目标，没有思想和纲领，那怎么样？终点就不存在了，方向也就不存在了，那两个人就容易朝着不同的方向。嗯，渐行渐远很神奇啊。当没有无外力的拉扯或者说诱惑的时候，共同的目标会让两个人自动的去做到行动上的一致。

什么叫行动一致啊？完整的说，我们在日常生活上的行动应该和上面提到的目标保持逻辑一致性。如果行动不一致，那么共同的目标不太容易实现，很容易被摧毁掉

- 比如说两个人的共同目标是大学毕业之后一起去另外一个城市读研。唉，那我们在学校的时候是不是要为这个目标做很多准备工作，对吧？得去学习，去考试，而如果对方天天在那玩着玩着，对方自己会模糊甚至忘记掉那个目标的，从而在心理上也否定掉那个目标。
- 俗称这个人变了，再像是说好了三年后结婚，结果对方天天和其他的异性接触，参加各种各样的局，甚至就出轨了，那一样的也会模糊掉原来那个目标
- 所以我们反推回去，**行动不一致**，很容易导致目标不一致。既然共同的目标是我们说的一段异地恋的前提，那**前提都没有了失败的几率肯定就高**
- 反过来我们看到了对方为了那个**共同目标做出了一致的行动**，那我觉得很好，这段异地恋就是有那个最**重要的成功的基础**的

:::

### 行动一致

行动一致会让 2 个人的日常生活紧扣那个共同目标。也让两个人相处更协调。尽可能的同频。

## 异地恋的特殊性和难点

- 生活更独立，更脱离于对方而存在
- 需求感动态平衡一一平淡
- 一方的需求感和投入度下降一一失衡
- 共同目标一一“特殊时期

::: tip

- 第一，两个人生活都更加独立，更脱离于对方而存在了，所以需求感的下降几乎是必然的
- 第二，如果需求感是动态平衡的，那也可以两个月没有理由分手，只是相处起来更平淡一些
- 第三，如果只有一方的需求感或者投入度下降，那这就会形成一种失衡，那么另一方会猜疑，会情绪化，会表现的没有安全感
- 第四，所以这个时候如果有共同目标的话，我们在心态上才能够产生一种特殊时期的感觉，才能够让我们去包容很多对方在表现中的瑕疵。

:::

::: details 举例

因为异地恋的形式很特殊，异地恋的**物理距离拉得很远**，两个人长期的部处在一个空间下，通常容易导致**某个人或者两个人的生活怎么样更加的独立**，更**脱离于对方而存在**。

比如极端情况，异国恋，大家每天接触到的人，需要做的事，看到的景色，一切都不一样的时候，很多细腻的、微妙的经历是很难传达的，

特别是通过两个人用的是非即时沟通，微信这种，只是在文字上简单的汇报一下

- 今天做了什么事，
- 吃了什么东西，
- 遇到了什么人，
- 考试过没过等等，
- 怎么样就是你现在心中想的那个词很无聊嘛？

如果连我们自己都觉得无聊，那时间久了对方渐渐的就会发现，唉，他自己好像没有那么需要我们，需要对方了。

这不是说我们哪句话没说好的问题，是因为这种汇报性质的沟通，做与不做其实没什么好大差别，而且有时候是真的会很累，回到家想在床上躺一下，看个视频放空，不行得和对方汇报。有时候多一事不如少一事，就出来了，很多情绪也开始自己消化掉了

- 这是需求感下降。需求感下降自然到投入也少了，依赖也少，这就是异地恋最特殊也最难的地方，也是几乎都会发生的情况。
- 但有一种情况，两个人都对方下降了需求感，那会好很多，因为总体上需求感还是一个平衡的，无非是感情变得很平淡。而如果双方能够接受这种平淡，又没有外界的诱惑和新的刺激的话，没有人来横刀夺爱的话，那其实也没有什么理由分手

**例子**

- 打个比方就说原本两个人对对方的需求度都是 80 分，因为异地两个人都要独自面对自己的世界，那么两个人同时把这种需求和投入降到了 40 分，可不可以？对，如果没有外界的干扰和自己其实是完全可以的，就平淡一点吧

:::

::: details 糟糕的

是一方需求感下降，另外一方却没有，或者说下降的没那么多，这就是失衡啊。

- 原来我们互相是 80 分，现在你降到 40，我还是 80，那就会变成我还是很需要你，很依赖你，但你却不那么需要我了。
- 具体表现就是我会天天想要找你聊天，问你在干嘛，去维持很多联系，但是你这边唉，好像越来越忙了，我也不知道你在忙什么，你也解释不上来，你觉得无非就是那些日常的琐事呗，说多了就没必要说了。那我会怎么样？
- 我会情绪化，我会开始猜疑表现的没有安全感。你作为需求感下降的那一方，看到我这种挣扎的状态，你会更加厌烦我，然后就没了嘛

很多没有第三者介入的异地恋都是这样分手的，其中的一方有了自己的独立的新世界，又没有一个统一的思想和纲领去把大家按在轨道上，就脱轨了嘛。

- 所以反过来，如果两个人能有一个一致的共同目标，有了行动指南，这就在无形中提醒我们的目的地是同一个地方，同一个方向，我们还是得朝着那个方向走。自然的，两个人更容易出现怎么样统一的行动？这个时候我们才会有更多的耐心和勇气去面对这些困难，也去包容对方表现中的瑕疵。
- 不会说因为对方一条消息回的慢，态度不那么热情，或者他今天很忙等等这种情况，而小题大做，相反很关键的这个时候我们会出现一种非常有助于异地恋的心态，也就是这些都是特殊时期里的特殊情况，而特殊时期是暂时是会过去的，当有了这种形态之后，熬的时候就不会觉得那么辛苦了，原本那些让人纠结的细节，杀伤力好像就没有那么强了

## 如何建立共同目标

::: warning

特殊时期的心态对我们自己去度过这段异地恋是非常非常重要的，但我要说这种心态不是凭空出现的，不是说我现在知道了有特殊时期的心态，我给自己打鸡血就能有的。这种心态的出现其实还是基于我们之前的大前提，共同的目标很神奇，特殊时期和共投目标是高度绑定的，当有了共同目标的时候，我们能不能看到终点，我们自动的，也就会相信眼下所处的困难是特殊时期

:::

::: tip
制定共同目标，记住这三个关键词，具体长期相信，当然说是设定，其实还是一种聊天，是我们在聊天的过程中去了解到的对方对于未来的愿景、理想规划等等这方面的信息，做信息交换的时候去做这个筛选

:::

目标得是具体的，目标得是长期的目标，两个人都得真的相信

- 共同目标是能够帮助我们去判断一段异地恋值不值得走下去的重要指标。
- 又是形成特殊时期心态的大前提

::: tip

- 第一，目标得是具体的，目标越是具体，我们越容易得出答案，应该怎么做？具体的目标也有助于我们将其拆分为更小可操作的步骤，才能够让我们真的去做到，也有执行力
- 第二，目标得是长期的目标，才能够形成思想和纲领【长期的目标，能够给我们更强大的去实现它的精神动力】
- 第三，**双方相信**两个人都得真的相信我们在商讨共投目标的时候是需要做筛选的，我们真正需要当做目标的不是我们单方面就像迫切实现的，而是我们两个人都愿意实现的那个相交的点，我们应该在聊天的时候抓到这个点去猛打，去强化这个目标。

:::
::: tip 目标

具体的目标才是具有可衡量性的，我们能够更容易的去评估和追踪进度。这能够帮助我们根据实际情况来调整方案，也提升实现目标的可能性嘛。

同时具体的目标有助于我们去把它拆分成更小的可操作的步骤，这会让我们更有执行力，也能够真正的去做到，而不是说说而已

:::

::: details 双方相信
**我信，我就会自我鼓励去完成这件事情，而如果我不信我能完成这件事情，自然我就很容易放弃**

个人真的相信的这个目标一定不是凭空而定的，对吧？

它得合理，得让两个人都相信，但凡有人在心里不是坚定的，相信这个目标一定会实现，对方很可能自己就咕咕了啊。

我，你这边需要拿鞭子抽他才能往前跑，相信会产生一种自我效能感，这也是心理学家阿尔伯特班杜拉提出的概念，

简单的说，我信，我就会自我鼓励去完成这件事情，而如果我不信我能完成这件事情，自然我就很容易放弃，对吧？

这就像是有的一些父母，

- 他经常批评小孩，说你不行的，你做不到的。
- 当这个小孩自己都不相信自己能做到，或者说他相信自己做不到的时候，他就不可能做到，

**所以相信很重要，并且一定是两个人的，因为只有我们单方面相信的，那不叫共创目标，那顶多算个愿望**

因为只有我们单方面相信的，那不叫共创目标，那顶多算个愿望。

**很多人去设定共同目标的时候会凭主观喜好，我喜欢的，我想要去实现的，我就当成目标。**

- 比如说好，我想要去某一个城市过安稳的日子，但对方是不是和你一样相信这个目标，对吧？

- 我们有可能忽略掉如果对方他确实想去那个城市，但他不一定想过安稳的生活，他想要出人头地，

这就是单方面的目标，这从性质上就会变成一愿望，而这种愿望是没有意义，如果对方不相信他无法从中获得自我效能感，他也不会有动力去做。

**忽略去设定共同目标的时候是要做筛选的，我们真的要去设定的不是我们单方面想要，而是两个人都坚信的那个相交的点**

- 需要去强化的在聊天当中不断提及的也仅仅是这个相交的点，对着这个部分我们应该去猛打，

- 而不是要强迫对方非要相信我们的那个愿望。这会导致对方可能为了照顾我们的情绪

**我怎么去判断对方是不是心里面真的相信，还是说口头上答应我呢**

- 简单，最直观、最可靠的就是我们上面说的，去看他的行动上和目标是否一致，因为一个人发自内心的坚定的在心里面去相信这个目标的时候，他在行动上一定是一致的

:::

我们小时候说长大了要当科学家一样，缩缩而已啦。
一个具体的目标应该是我要当科学家，

- 分数线是多少岁的时候达到了研究院的入职标准？

结婚了以后

  我们越是能够容易得出这个答案，应该怎么做

:::

::: details 长期例子
下个月我要去你的城市找你约会，这不是长期的，这很难成为一个有足够引导力的目标，它最多是一个小计划，而小的计划通常不会涉及到价值观，所以引导力也很弱。

而长期的目标才能够形成思想和纲领。

比如为什么我要认真读书啊？是因为我要考试，考完试我一页书都不会看的。但周恩来说为中华崛起而读书，这很明显涉及到价值观了。
怎么样？他引导力是很强的，那不管我是贫穷或是富裕，我身处何地，我会一直读书的。

所以两个人在一起异地恋，不妨把这个目标往长期的那个区域去看一看，

- 像是组成一个什么样的家庭，
- 未来过一种怎样的生活
- 什么生活方式，
- 包括结婚或者去到某个城市定居。

这些都算长期的目标，能够给我们更强大的去实现它的精神动力

## “特殊时期”的三元素怎么调整？

1. 氛围感：核心思路是重质不重量【维持住氛围水平，每一次的互动都做得更有意思，而线上的互动的质量又从哪里来？OK，这里包含，但不限于多用】

::: tip 可以做的事情

- 培养 1- 2 个共同的爱好，去增加共同的话题，
- 节日和纪念日，也可以通过视频或者电话去约会，要有仪式感，
- 偶尔的准备一些礼物或者是惊喜去送给对方
- 同一时段同时看一部电影
- 两个人一起下楼去跑步

2. 亲密感，多元化的方式去调动亲密的欲望，【目的：把这种距离削弱掉的亲密感给维持住】

::: tip 可以做的事情

- 发好看的、能展示我们吸引力的照片、视频啊。当然，注意保护隐私

- 多聊一些情侣之间的私密话题幻想，
- 即便暂时见不到面, 也可以在聊天中分享一下一次见面希望做什么，
- 再比如分享比较能够刺激欲望的电影我们前面说的可以同频一起在线看边聊
- 然后再是通过文字和语音信息去展示性别的吸引力

线上激发出了对方的欲望，然后他又没有办法和我亲密接触，会不会造成他更有可能出轨？

结论，影响很小，因为本质上性唤起和出轨之间并不是直接的因果联系

- 这是两个各自独立的概念，就是说性唤起是否会导致一个人直接出轨，如果会的话，那完了，我们天天在网上都会看到各种各样的照片、视频，都会产生新唤起，那是不是这个人他天天都要出轨，
- 所以反推过来不成立啊？是当一个人觉得你看这网络上的这些照片多好多棒，如果他说我对象多好，我对象哪哪都不好的时候，才容易导致出轨

我们发自己好看的照片了，把新唤起的对象导向自己，这会增加对方对我们的念想，是不是也会激发对方和我们见面的欲望？

- 另一方面退一步说，唤起对象是我们自己总比别人要好，并且这个时候共同目标的作用就体现出来了，我们说一致的目标导致一致的行为，
- 如果两个人的共同目标就是长久了，长期的在一起，那眼下一起守身如玉，抵制诱惑，一起不做什么也是一致的行为

3. 走心：打算和对方聊心事之前，先试探【打算和对方聊心事之前，包括我们想问对方的心事的时候，最好先试探一下】

::: tip 走心
我们的注意力和重心应该放在电话或者视频这种聊天方式，这种即时沟通的聊天方式上，因为这些形式能够更加准确地传递出我们和对方的心情和情绪

- 如果我们想和对方聊心事之前，最好先试探一下，比如我说我今天遇到些事，我现在不太开心，如果对方很积极的回应，问你怎么了？让我们好好跟他说说，那就说明他对方的状态和环境现在通常还不错，情绪能量也比较高，大概是能够接得住我的情绪的，这就是个好时机。

- 反过来对方的语气语调很低落，环境都是一片嘈杂，或者是非即时沟通的时候，对方隔了很长时间再回复，那就意味着这个天时地利人和并没有达到，对方并没有在一个适合走形、适合深度对话的环境下面。那这个时候我们最好能收住，就不要硬把情绪灌给对方了，反正他也接不住，反而还给对方造成压力。

- 如果我们有那个特殊时期的心态的话，其实我们是可以自己消化掉的，不管这是否让你觉得委屈，我要说事实就是这样，为了更好的度过异地恋，达到那个终点，这是我们在异地恋中必须要克服的事情

:::

因为不处在同一空间下，所以很自然的两个人的交流的形式往往也变得单一了，无非是微信、电话、视频，也缺乏了约会来制造共同的精力，很容易把聊天变成一种没话找话，一种完成任务的状态，结果就是氛围水平降低了，聊天也寡淡无味了。

:::
